# Система классификации пожарной опасности

Веб-приложение для прогнозирования класса пожарной опасности на основе метеорологических данных и индексов FWI. Учебный проект по дисциплине "Применение методов машинного обучения в информационно-обоснованных решениях", МГТУ СТАНКИН.

## Функциональность
1. Оперативный расчет класса опасности по вводимым параметрам
2. Пакетный анализ загружаемых CSV-файлов с визуализацией результатов
3. Отображение метрик качества модели и важности признаков

## Технологии

- Python 3.13
- scikit-learn (Random Forest)
- pandas, numpy
- matplotlib, seaborn
- HTML/CSS (Jinja2)

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/VikaMinkina/ProjectWork_Lab4.git
cd ProjectWork_Lab4
```
2. Установите зависимости:
```bash
pip install -r requirements.txt
```
3. Убедитесь, что файлы модели находятся в корне проекта:
fire_model.pkl
scaler.pkl
encoders.pkl
4.Запустите приложение:
```bash
python main.py
```
5. Откройте браузер и перейдите по адресу http://127.0.0.1:5000
